Creating and pushing to github repository


Ok here we go! Let's make a repository! Please navigate to your github. Log in and you should see your main page.

Click the green New button in the left aside bar. Enter your repository name. Make it something useful that pertains to your content. Don't name a website about potatos something like spam. Put in a description, make sure it is public and check "Add a README file." Finally, click the green "Create repository."
